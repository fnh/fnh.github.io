Web Components

A set of well-supported specs:

  • Custom Elements
  • HTML Templates
  • Shadow DOM
  • HTML Imports

So, why don't we just use the platform?

  • Quite some boiler plate code necessary
  • Developer experience could be a lot smoother
  • Not straight forward to build

Stencil JS

  • Build time tool for developing 100% standard-compliant plain JavaScript web components
  • Borrows ideas from popular frameworks without itself being yet another one
  • Utilizes: TypeScript, JSX / TSX, SCSS
  • Quick to learn

DEMO TIME!


								import { Component, Prop } from '@stencil/core';
								
								@Component({
								  tag: 'unkonf-greeter'
								})
								export class UnkonfGreeter {
								  @Prop() first: string;
								  @Prop() last: string;
								  
								  render() {
									return (<div>Hello, {this.first} {this.last}</div>);
								  }
								}
